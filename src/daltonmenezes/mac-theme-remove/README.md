# Mac Theme Remove

This recipe will remove:

- MacBuntu icons
- MacBuntu theme
- MacOS X Sierra wallpaper
- Plank dock
- Gnosierra Plank theme
- Albert spotlight

# Usage

| Command | Support |
| --- | --- |
| `uni bake mac-theme-remove` | Ubuntu 16.04 (Unity) |
