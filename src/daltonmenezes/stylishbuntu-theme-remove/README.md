# Stylishbuntu Theme Remove

This recipe will remove:

- El General icons
- Flatabulous theme

# Usage

| Command | Support |
| --- | --- |
| `uni bake daltonmenezes/stylishbuntu-theme-remove` | Ubuntu 16.04 (Unity) |
