# Mac Theme

This recipe will install and configure:

- MacBuntu icons
- MacBuntu theme
- MacOS X Sierra wallpaper
- Plank dock
- Gnosierra Plank theme
- Albert spotlight `{Shift + Space}` for use it.

# Usage

| Command | Support |
| --- | --- |
| `uni bake mac-theme` | Ubuntu 16.04 (Unity) |

# Uninstallation

| Command |
| --- |
| `uni bake mac-theme-remove` |
