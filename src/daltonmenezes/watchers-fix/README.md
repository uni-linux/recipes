# Watchers Fix

This recipe will fix the `ERROR: ENOSPC: SYSTEM LIMIT FOR NUMBER OF FILE WATCHERS REACHED` issue.

# Usage

| Command | Support |
| --- | --- |
| `uni bake watchers-fix` | Ubuntu and based distributions |
